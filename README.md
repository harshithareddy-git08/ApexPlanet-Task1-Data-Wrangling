# 📊 Sales Analytics  Project | ApexPlanet Internship

## 📌 Project Overview

This project focuses on analyzing sales data using Python and Google Colab to extract meaningful business insights. The project covers the complete data analytics workflow, including data cleaning, preprocessing, feature engineering, exploratory data analysis (EDA), and visualization.

The goal is to transform raw sales data into actionable insights that can support better business decision-making.

---

## 🎯 Project Objectives

- Clean and preprocess raw sales data
- Handle missing values and duplicate records
- Create meaningful business features
- Perform exploratory data analysis (EDA)
- Visualize sales trends and customer behavior
- Generate actionable business insights

---

## 📂 Dataset Information

### Raw Dataset

**File:** `ApexPlanet_DataAnalytics_Dataset.xlsx`

Contains sales transaction data including customer, product, and revenue-related information.

- Total Records: 1000
- Features: 12

### Cleaned Dataset

**File:** `Cleaned_Sales_Dataset.xlsx`

After preprocessing:

- Total Records: 981
- Features: 18

### Data Dictionary

**File:** `Data_Dictionary.xlsx`

Contains:

- Column Names
- Data Types
- Feature Descriptions
- Business Definitions

## 🔄 Project Workflow

### 1. Data Loading

Loaded the raw sales dataset into Google Colab using Pandas.

### 2. Data Cleaning

Performed the following preprocessing steps:

- Handled missing values
- Removed duplicate records
- Corrected data types
- Validated dataset consistency

### 3. Feature Engineering

Created additional business-focused features:

- Month
- Year
- Quarter
- Revenue Segment
- Age Group
- High Value Customer

### 4. Exploratory Data Analysis (EDA)

Analyzed:

- Monthly Sales Trends
- Quarterly Sales Trends
- Revenue Distribution
- Customer Behavior
- Product Performance
- Customer Segmentation

### 5. Data Visualization

Created visualizations using Matplotlib and Seaborn to identify trends, patterns, and business opportunities.

---

## 📈 Key Insights

### Sales Performance

- Sales varied across different months and quarters.
- Certain periods generated significantly higher revenue.

### Customer Insights

- High-value customers contributed a major share of revenue.
- Different age groups showed varying purchasing behavior.

### Product Insights

- A small number of products generated the majority of sales.
- Product performance differed across categories.

### Revenue Insights

- Revenue segmentation helped identify profitable customer groups.
- High-revenue segments contributed significantly to overall business growth.
